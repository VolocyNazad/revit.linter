
## 🎯 Уровни предупреждений

| Уровень               | Значение                                              | Рекомендация для пользователя          |
| :-------------------- | :---------------------------------------------------- | :------------------------------------- |
| ❌ **Ошибка**          | Критичная проблема, влияющая на корректность проекта. | Обязательно исправить.                 |
| ⚠️ **Предупреждение** | Потенциальная проблема, требующая внимания.           | Рекомендуется исправить или проверить. |
| 💬 **Сообщение**      | Информационное сообщение без строгих требований.      | Ознакомиться по желанию.               |
