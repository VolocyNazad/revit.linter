# ⚙️ Параметры

| Свойство                | Тип                                           | Описание                                                                                                                                    | Пример                                                                                                                                               |
| :---------------------- | :-------------------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------ | :--------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Code**                | `string`                                      | Уникальный идентификатор диагностики в формате `[Префикс][Номер]`. Префикс указывает на категорию проверки.                                 | `SHRD001`                                                                                                                                            |
| **Description**         | `string`                                      | Краткое описание того, что проверяет диагностика.                                                                                           | `Проверка семейств на их использование в документе.`                                                                                                 |
| **MessageFormat**       | `string`                                      | Шаблон сообщения, выводимого пользователю. Поддерживает плейсхолдеры в фигурных скобках `{ }` для подстановки значений во время выполнения. | `Элемент с именем` [[diagnostic-message-placeholders\|'{elementName}']] `и идентификатором` [[diagnostic-message-placeholders\|'{elementId}']] `...` |
| **DefaultSeverity**     | [[diagnostic-severity\|`DiagnosticSeverity`]] | Уровень важности по умолчанию. Пользователь может изменить его в настройках.                                                                | `DiagnosticSeverity.Warning`                                                                                                                         |
| **IsActive**            | `bool`                                        | Включена ли диагностика по умолчанию. Если `false` — она не выполняется, пока пользователь явно не активирует её.                           | `true`                                                                                                                                               |
| **IsObsolete**          | `bool`                                        | Помечена ли диагностика как устаревшая. Устаревшие проверки могут быть удалены в будущих версиях.                                           | `false`                                                                                                                                              |
| **ObsoleteDescription** | `string`                                      | Пояснение к устареванию. Содержит информацию о замене или причине удаления. Пустая строка, если диагностика актуальна.                      | `Используйте SHRD003 вместо этой проверки.`                                                                                                          |
