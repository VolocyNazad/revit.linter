
В Revit **"Dockable panes" (закрепляемые панели)** — это элементы интерфейса, которые можно свободно перемещать, закреплять у краев главного окна программы или делать плавающими (включая размещение на дополнительных мониторах). Это позволяет гибко настраивать рабочее пространство под свои привычки.
